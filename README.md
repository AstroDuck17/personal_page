# Personal Portfolio Website

A clean, professional portfolio website showcasing my experience, projects, and skills.

## Structure

- **about.html** - Landing page with profile and introduction
- **resume.html** - Experience, education, and technical skills
- **projects.html** - Showcase of key projects
- **contact.html** - Contact form
- **new-styles.css** - Main stylesheet
- **images/** - Profile photo and assets

## Design

- **Color Scheme**: Green accents (#0A6522), beige background (#E8DCC8), white cards
- **Typography**: Clean, modern sans-serif
- **Layout**: Responsive grid-based design
- **Navigation**: Consistent header/footer across all pages

## Features

- Responsive design for all screen sizes
- Clean, professional UI
- Downloadable CV
- GitHub and LinkedIn integration
- Contact form with email functionality

## Pages

### About
Introduction and profile card with quick links to resume and projects.

### Resume
Detailed view of:
- Work experience
- Education and achievements
- Technical skills (Languages, Protocols, Tools)

### Projects
Featured projects:
- Network File System
- Audio-Guided Microbot
- C-Shell

### Contact
Professional contact form with email integration.
